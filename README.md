# CCNA Engineering Residency

> A 60-day network engineering residency focused on designing, deploying, troubleshooting, validating, and documenting enterprise networks using Cisco technologies.

---

## Overview

The purpose of this repository is to simulate the daily responsibilities of a Network Engineer through a structured 60-day engineering residency.

Rather than studying solely for the Cisco Certified Network Associate (CCNA) exam, this project emphasizes the practical skills required to deploy and support enterprise networks. Every lab is approached as a real engineering work item and is documented using industry-inspired processes.

Each day includes planning, implementation, validation, troubleshooting, documentation, and version control to mirror professional engineering workflows.

---

## Residency Objectives

By the end of this residency, I will be able to:

- Design small to medium-sized enterprise networks
- Configure Cisco routers and switches
- Deploy VLANs and trunk links
- Configure Inter-VLAN Routing
- Implement Static and Dynamic Routing
- Configure OSPF
- Deploy DHCP, NAT, and ACLs
- Troubleshoot Layer 1 through Layer 3 issues
- Document network changes using professional engineering practices
- Build a portfolio demonstrating real-world networking skills

---

## Engineering Philosophy

This residency is built around one principle:

> Every lab should be treated like a production network deployment.

Every task includes:

- Planning
- Documentation
- Configuration
- Validation
- Troubleshooting
- Lessons Learned
- Version Control

The objective is to develop engineering habits rather than simply memorize commands.

## Engineering Workflow

Every deployment follows the same workflow:

1. Review engineering ticket
2. Study the required concepts
3. Design the network
4. Configure devices
5. Verify functionality
6. Troubleshoot issues
7. Document the deployment
8. Record a walkthrough
9. Commit changes to Git
10. Push updates to GitHub

---

## Documentation Standards

Every deployment includes:

- Engineering Ticket
- Lab Report
- Engineering Journal
- Configuration Backup
- Verification Results
- Network Diagram
- Lessons Learned
