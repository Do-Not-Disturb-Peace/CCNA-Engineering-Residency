# CCNA Engineering Residency

A 60-day hands-on networking residency designed to build CCNA-level knowledge through realistic Cisco Packet Tracer deployments, troubleshooting tickets, technical documentation, and portfolio-ready presentations.

## Goals

- Prepare for the Cisco CCNA 200-301 exam.
- Build practical junior network engineer skills.
- Design, configure, verify, break, and troubleshoot enterprise-style networks.
- Document work using Markdown, Git, GitHub, diagrams, and screen recordings.
- Produce a public technical portfolio suitable for LinkedIn and job applications.

## Core Resources

- Cisco CCNA 200-301 Official Cert Guide
- Kevin Wallace: Cisco Certified Network Associate (CCNA) v1.1 Cert Prep
- Cisco Packet Tracer
- Included hands-on Packet Tracer laboratory files
- Visual Studio Code
- Git and GitHub
- OBS Studio
- diagrams.net

## Daily Workflow

1. Read the assigned Official Cert Guide topics.
2. Watch the matching Kevin Wallace course sections.
3. Review the business ticket and change request.
4. Design the topology and addressing plan.
5. Build the deployment in Packet Tracer.
6. Verify the required outcomes.
7. Introduce a fault and troubleshoot it.
8. Complete the lab report and engineering journal.
9. Record a short private walkthrough.
10. Commit and push the completed work.

## Repository Structure

- `labs/` — Daily tickets, Packet Tracer files, configurations, and verification evidence
- `templates/` — Reusable documentation templates
- `journal/` — Network Engineering Journal entries
- `weekly-reviews/` — Weekly retrospectives and technical reviews
- `diagrams/` — diagrams.net source files
- `images/` — Exported topology images and screenshots
- `configurations/` — Device configuration backups
- `verification/` — Command output and test evidence
- `final-project/` — Multi-site enterprise capstone
- `docs/` — Roadmap, standards, and setup instructions

## Public Portfolio Policy

Do not publish:

- Real passwords or secrets
- Personal email addresses
- Employer-sensitive information
- Copyrighted course PDFs, book scans, or paid exercise material
- Large raw screen recordings

Publish only original diagrams, configurations, notes, screenshots, and Packet Tracer work that you are permitted to share.
